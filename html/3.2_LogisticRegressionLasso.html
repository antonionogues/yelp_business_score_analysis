0. Introduccin

El Modelo de Regresin Logstica es un modelo de machine learning utilizado para predecir el resultado de una variable categrica (variable binaria) a partir de un conjunto de variables explicativas. A diferencia de la regresin lineal, que se utiliza para predecir valores continuos, la regresin logstica se utiliza para predecir la probabilidad de que un restaurante tenga buena puntuacin o no en funcin de sus atributos.

6.3. Curva ROC y Lift¶

Para evaluar nuestro modelo, las representaremos graficamente para deducir que tan bueno es el modelo.

In [14]:
# keep probabilities for the positive outcome only
yhat = predict_probabilities[:, 1]

fpr, tpr, thresholds = roc_curve(y_val, yhat)

plt.plot([0,1], [0,1], linestyle='--', label='No Skill')
plt.plot(fpr, tpr, marker='.', label='Logistic Regression')
# axis labels
plt.xlabel('False Positive Rate')
plt.ylabel('True Positive Rate')
plt.legend()
# show the plot
plt.show()

# Display the ROC AUC.

print('The Compute Area Under the Receiver Operating Characteristic is '+ str(roc_auc_score(y_val,yhat)))
