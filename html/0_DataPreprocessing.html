2. Introduccin

Para la prctica final de la asignatura, trabajaremos con el dataset pblico de la compaa Yelp, el cual se puede descargar aqu.

Yelp es una empresa estadounidense que ofrece una plataforma en lnea para revisar y recomendar servicios locales, como restaurantes, hoteles y spas. La empresa tambin ofrece una aplicacin mvil para facilitar el acceso a estas reseas y recomendaciones. Yelp se fund en 2004 y desde entonces ha crecido para convertirse en una de las principales plataformas de reseas en lnea a nivel mundial. La empresa se dedica a ayudar a las personas a encontrar los mejores servicios locales y hacer negocios con ellos.

2.1. Objetivo de negocio¶

El objetivo de negocio es a travs de la informacin registrada por los negocios en la plataforma y sus caractersticas, poder predecir si el promedio de puntuacin que alcanzarn por parte de los usuarios ser alto o bajo.

Para ello ser necesario realizar un proceso de transformacin de datos, necesario para poder tratar de una forma sencilla las variables durante el anlisis exploratorio.

3. Yelp Dataset

3.1. Información del dataset¶

El dataset de Yelp est compuesto por 5 archivos de tipo JSON:

- yelp_academic_dataset_business.json

- yelp_academic_dataset_checkin.json

- yelp_academic_dataset_review.json

- yelp_academic_dataset_tip.json

- yelp_academic_dataset_user.json

Cada archivo contiene variables y observaciones acerca de diferentes aspectos: negocios adscritos, visitas al negocio, reseas, sugerencias y usuarios respectivamente. Si se quisiera consultar ms informacin acerca de la estructura de cada archivo y variables incluidas en los archivos, se puede encontrar en el archivo Dictionary.md, el cual se encuentra en el directorio data/processed.

3.2. Características del dataset¶

El peso del dataset descomprimido es de 8.65 GB y como ya se coment en el punto anterior en formato JSON.

Para generar ideas de negocio y aplicar posteriormente modelos de Machine Learning ser importante en un primer lugar realizar una transforamcin de los datos, con el objetivo de reducir la carga computacional y conseguir una mayor sencillez trabajando con ellos.

4. Transformacin del formato de ficheros

El formato JSON (JavaScript Object Notation) es un formato de texto que se utiliza para transmitir datos en una estructura de objetos anidados. Es un formato comnmente utilizado para intercambiar datos entre aplicaciones web.

Sin embargo, existen formatos mucho ms eficientes en trminos de lectura y escritura, especialmente en ciencia de datos, como el formato Parquet. Parquet es un formato de archivo de columnas que se utiliza para almacenar grandes cantidades de datos tabulares.

Es por ello que procederemos a convertir los archivos descargados en formato Parquet.
