0. Introduccin

En este documento, nos centraremos en la creacin de un Modelo Light GBM.

LightGBM es un mtodo de ensamblaje de refuerzo de gradientes que se basa en rboles de decisin. Al igual que con otros mtodos basados en rboles de decisin, LightGBM se puede utilizar tanto para la clasificacin como para la regresin. LightGBM est optimizado para un alto rendimiento con sistemas distribuidos.

Una vez creado el modelo, procederemos a optimizarlo con los valores de train y predecir los valores de validacin.

3. Creacin del modelo

Para la creacin del modelo LightGBM utilizaremos el algoritmo LGBMClassifier de la libreria lightgbm.

Creamos un Pipeline que ejecute el preprocesado primero y el algoritmo LightGBM despus.

Para el algoritmo designaremos los parmetros por default, pero utilizaremos todo el procesador para agilizar el proceso.

In [5]:
# Design the parameters.

n_jobs = -1
random_state = 12345

# Create a pipeline that applies a preprocess of the data and then the LogisticRegression model.

lightgbm_model = Pipeline([
    ('preprocessor', preprocessor),
    ('clasificador', lgb.LGBMClassifier(n_jobs=n_jobs, random_state=random_state)
    )
])

6.3. Curva ROC y Lift¶

Para evaluar nuestro modelo, la representaremos graficamente para deducir que tan bueno es el modelo.

In [28]:
# Keep probabilities for the positive outcome only
yhat = predict_probabilities[:, 1]
# Calculate roc curves
fpr, tpr, thresholds = roc_curve(y_val, yhat)

plt.plot([0,1], [0,1], linestyle='--', label='No Skill')
plt.plot(fpr, tpr, marker='.', label='Light GBM')
# Axis labels
plt.xlabel('False Positive Rate')
plt.ylabel('True Positive Rate')
plt.legend()
# Show the plot
plt.show()

# Display the ROC AUC.

print('The Compute Area Under the Receiver Operating Characteristic is '+ str(roc_auc_score(y_val,yhat)))
